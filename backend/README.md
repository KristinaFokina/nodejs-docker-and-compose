# Backend КупиПодариДай

_КупиПодариДай_ - это веб-приложение, которое позволяет пользователям создавать списки своих пожеланий и делиться ими с друзьями, или скидываться на подарки другим пользователям.

## Инструкция

1. Убедиться, что на вашем компьютере установлены Node.js и PostgreSQL.
2. Установите зависимости, запустив команду `npm install`.
3. Создайте базу данных PostgreSQL с названием `kupipodariday`.
4. Запустите сервер, выполнив команду `npm run start`.
5. Откройте веб-браузер и перейдите по адресу `http://localhost:3000`.
